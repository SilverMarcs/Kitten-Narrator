# Kitten Narrator

A text-to-speech narration app for iOS. Add articles, PDFs, or text and listen to them with natural-sounding voices.

## Features

- Import content from text, URLs, PDFs, or clipboard
- Multiple voice options with preview
- Background audio playback with speed control
- Library with search and filtering
- Share Extension for adding content from other apps
- SwiftData persistence
